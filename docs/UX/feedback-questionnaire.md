## Google Form Setup Notes

When creating the Google Form:

1. **Sections**: Use page breaks between sections 1-7
2. **Required fields**: Make Q1-Q6, Q8, Q11, Q15-Q18, Q25-Q29, Q31-Q32 required
3. **Conditional logic**: Show Section 7 only if Q1 = "Subject-Matter Expert"
4. **Likert scales**: Use the "Linear scale" question type (1-5)
5. **Multiple choice**: Use "Multiple choice" for single-select, "Checkboxes" for multi-select
6. **Open text**: Use "Long answer" for descriptive questions, "Short answer" for brief ones
7. **Response collection**: Collect email addresses if you want to follow up
